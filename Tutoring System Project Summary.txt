# Tutoring System Project Summary

## Project Overview
โปรแกรมสถาบันกวดวิชา English Mania

## Key Features
1. การจัดการปฏิทินที่ยืดหยุ่น ตามสิทธิ์มาใช้งาน สามารถ เลือกข้อมูลได้ตามต้องการเช่น ติวเตอร์, นักเรียน, ห้องเรียน
2. การจัดการข้อมูลนักเรียน สามารถแสดงประวัติต่างๆ เช่น การเข้าเรียน, คอร์สที่เรียนไปแล้ว, การจ่ายเงิน, สมุดพกและดาวโหลดเอกสารต่างๆ ที่เกี่ยวข้องกับตัวเองได้ เช่น สมุดพก หรือ ประกาศษณียบัตรเป็นต้น
3. การจัดการข้อมูลติวเตอร์ สามารถแสดง ดูประวัติการสอนแยกตามนักเรียนและคอร์สได้, รายงานรายได้ของตัวเอง
4. การสร้างคอร์สที่ยืดหยุ่น เช่น คอร์สรายชั่วโมง-หักเป็นชัวโมง และสามารถเลื่อนเวลาเรียนได้, คอร์สรายเดือน-หักตามวันสามารถสลับวันเรียนได้, คอร์สรายเทอม-เลื่อนหรือสลับวันไม่ได้
5. การบันทึกชั่วโมงการเข้าเรียนผ่านทาง line โดยระบบเป็นคนส่งผ่าน line ให้นักเรียนอัตโนมัติ เมื่อนักเรียนกดยืนยันและติวเตอร์ส่งฟอร์มบันทึกการสอนของนักเรียนในการเรียนวันนั้นๆ แล้วจึงบันทึกว่าเข้าเรียนแล้ว แต่ admin สามารถแก้ไขทีหลังได้
6. รายงานต่างๆ และสรุปรายงานทางบัญชี
7. นักเรียน และ ติวเตอร์ สามารถใช้บริการผ่าน web และ ดูข้อมูลอย่างเดียวผ่าน line ได้
8. สรุปปฎิทิน ชั่วโมงที่เปิดให้ลงทะเบียนเรียน ตามคิวว่างของติวเตอร์ โดยสามารถเลือกจาก คอร์สที่สนใจแล้วจะแสดง ชั่วโมงที่ยังสามารถลงทะเบียนได้
9. การแจ้งเตือนต่างๆ ผ่าน line และ email
10. พิพม์ใบแจ้งหนี้ ใบเสร็จในรูปแบบ pdf ได้
11. export รายงานต่างในรูปแบบ csv ได้
